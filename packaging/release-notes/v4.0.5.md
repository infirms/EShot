## EShot v4.0.5

This release improves the GIF and video recording workflow and fixes several KDE Plasma integration issues.

### Changes

- Redesigned the recording controls to match the screenshot toolbar
- Added responsive placement below, above or inside the selected recording area
- Kept recording controls and borders out of the captured video or GIF
- Added pause and resume support to GIF recording
- Excluded paused time from GIF duration and recording limits
- Fixed saved-file notifications on KDE Plasma so clicking them opens the containing folder
- Prevented localization tests from overwriting the user's saved application language
- Completed the new recording and notification translations in all eight interface languages

Linux support is currently tested on KDE Plasma 6 with Wayland. GNOME, Cinnamon and other desktop environments are not supported yet and may not work correctly.
