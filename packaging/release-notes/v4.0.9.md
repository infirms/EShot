## EShot v4.0.9

This release adds faster window capture on Windows and improves the recording setup experience.

### Window capture

- Added a dedicated Window Capture mode on Windows
- Use `Shift + Print Screen` by default or start it from the tray menu
- Added smooth animated window highlighting
- Kept regular `Print Screen` focused on free region selection
- Made the Window Capture shortcut configurable
- Added current shortcut hints to the tray capture actions

### Recording and Windows fixes

- Enabled desktop and microphone audio by default for new recording settings
- Improved disabled audio controls and recording FPS options
- Improved Windows audio device detection for MinGW builds
- Prevented the settings window from opening unexpectedly during tray startup
- Added release validation for the Qt SVG runtime required by toolbar icons

Window Capture is currently available on Windows. KDE Plasma and GNOME window capture require desktop-specific integration and are planned separately.
