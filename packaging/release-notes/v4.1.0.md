## EShot v4.1.0

This release improves capture reliability and desktop integration on KDE Plasma and GNOME.

### Capture and annotation fixes

- Restored capture panel keyboard focus after closing upload and OCR dialogs
- Fixed `Ctrl+C` no longer working after the upload dialog was closed
- Fixed the annotation color picker freezing on Linux
- Reduced capture panel prewarm delay for faster startup

### GNOME improvements

- Fixed a first-run race that could prevent the Print Screen shortcut from registering
- Correctly handles portals that already know EShot's application ID
- Added the AppIndicator extension to CachyOS and Arch GNOME runtime setup
- Added the GNOME tray extension as an optional AUR dependency

### Interface

- Fixed settings briefly opening with the wrong theme
- Applied light, dark and high contrast palettes consistently from startup

GNOME support is still experimental and desktop behavior may vary between GNOME Shell and portal versions.
