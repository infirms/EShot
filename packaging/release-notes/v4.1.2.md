## EShot v4.1.2

This maintenance release improves localization, visual search reliability, and upload services.

### Improvements

- Completed the affected Settings translations across all eight supported languages
- Made the Reset, Cancel, and Save buttons use consistent sizing across translations
- Added automatic visual search upload fallback through Catbox, Uguu, and TmpFiles
- Added clearer localized errors when every visual search upload provider is unavailable
- Restored Yandex Disk in the upload provider list
- Updated the Windows dependency cache key to account for runner and vcpkg changes

### Fixes

- Fixed Google Lens and Yandex Images searches failing when the previous temporary host was unavailable
- Fixed TmpFiles links opening an HTML landing page instead of the uploaded image
- Removed Litterbox from the provider list because its upload service is no longer reachable
